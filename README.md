# 🎧 Music Bridge Library
A Java-based web server for managing your music and podcasts.

## Features
* Add Songs and Podcasts via a web interface.
* Search through your library in real-time.
* Favorite and Delete tracks.
* Data is saved locally in `music_bridge.dat`.

## How to Run
1. Compile: `javac Experiments/miniproject.java`
2. Run: `java Experiments.miniproject`
3. Visit: `http://localhost:8085`
